# TASK1-Calculator-CLI-App(Elevate-Labs)
This project is a Python CLI Calculator that supports basic arithmetic operations. It demonstrates core programming concepts including functions, loops, conditionals, and error handling. Designed for beginners to practice Python fundamentals and command-line interaction.               

The application is built using fundamental Python concepts such as functions, loops, conditionals, and exception handling, making it an excellent beginner-friendly task.            

Users can interact with the calculator directly through the terminal, select operations from a menu, input numbers, and receive results instantly. The program continues running in a loop until the user chooses to exit. This project helps beginners strengthen their understanding of Python logic flow, CLI interaction.
